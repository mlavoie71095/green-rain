GreenRainReplacer - Build with Visual Studio Code
=================================================

Requirements:
- .NET 6 SDK or later: https://dotnet.microsoft.com/en-us/download
- SMAPI installed: https://smapi.io/
- VS Code with C# extension (by OmniSharp)

Instructions:

1. Copy the required SMAPI files into a folder at:
   GreenRainReplacer/SMAPI/

   Required files:
   - StardewModdingAPI.dll
   - StardewValley.dll

2. Open this folder in Visual Studio Code.

3. Run the following command to build the mod:
   dotnet build

4. The compiled GreenRainReplacer.dll will appear in:
   Mods/GreenRainReplacer/

5. Move the entire output folder into your Stardew Valley Mods directory.
